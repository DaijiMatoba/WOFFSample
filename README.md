# WOFF アプリ サンプル

このリポジトリはWOFFアプリのサンプルプロジェクトを含んでいます。

## 概要

- ユーザープロフィール情報の表示
- Geolocation APIを使用した位置情報の取得
- QRコードのスキャン
- アプリの終了機能

## 使用方法

詳しい使用方法については、[こちらのページ](https://qiita.com/iwaohig/items/57e201a6dbaa72cae61f)をご参照ください。

## 必要なライブラリや外部リソース

- Google Fonts: Orbitron
- WOFF SDK
